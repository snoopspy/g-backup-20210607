## Install

### Linux
* Run ./deps.sh in command line.
* Install Qt.
* Run the follwing command.
  ```
  make
  make distclean
  ```

### Android
* Run the following commands.
  ```
  source ../android-build/android.profile
  ./android-build.sh
  make distclean
  ```
